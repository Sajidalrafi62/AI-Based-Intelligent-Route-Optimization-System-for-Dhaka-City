# AI-Based-Intelligent-Route-Optimization-System-for-Dhaka-City
This is a complete intelligent routing system for Dhaka city that extracts real map data, applies multiple search algorithms, and visualizes optimal routes through an interactive UI.

To run                                                    
                                                                                                                                                                         
  # 1. Install dependencies
  pip install -r requirements.txt                                                                                                                                                                                                                                                                                    
                                                            
  # 2. Test full CLI pipeline                                                                                                                                            
  python3 main.py --src Shahbagh --dst "Gulshan 1" --preset balanced
                                                                                                                                                                         
  # 3. Launch the UI                                        
  streamlit run app/streamlit_app.py
